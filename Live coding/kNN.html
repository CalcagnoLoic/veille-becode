Le kNN ou les k plus proches voisins¶

Rappel¶

Attribuer une "tiquette" a une nouvelle donnes en fonction de son voisinage le plus proche.

Le voisinage est dtermin via l'hyperparamtre k grce une validation croise.

Présentation du problème¶

Nous allons travailler avec un jeu de donnes qui contient des informations physico-chimiques de vins portugais, ainsi que leur qualit telle que note par des humains.

Le problme, sur ces donnes, est de prdire automatiquement la qualit sur la base de ces informations, afin dassister le travail dvaluation des nologues, damliorer la production de vin, et de cibler le got des consommateurs sur des marchs de niche.

from sklearn import neighbors, metrics

# Fixe des valeurs d'hyperparamtres 
param_grid = {'n_neighbors':[3,5,7,9,11,13,15]}

# Choix du score  optimiser => proportion de prdictions correctes
score = 'accuracy'

# Classifieur kNN
clf = model_selection.GridSearchCV(
    neighbors.KNeighborsClassifier(),
    param_grid,
    cv = 5,
    scoring = score)

# Optimisation du classifieur sur le training set
clf.fit(X_train_std, y_train)

# Affichage du meilleur hyperparamtre 
print("Meilleur hyperparamtre :") 
print(clf.best_params_)

# Afficher les performances
print("Rsultats de la validation croise :")
for mean, std, params in zip(
        clf.cv_results_['mean_test_score'], # score moyen
        clf.cv_results_['std_test_score'],  # cart-type du score
        clf.cv_results_['params']           # valeur de l'hyperparamtre
    ):

    print("{} = {:.3f} (+/-{:.03f}) for {}".format(
        score,
        mean,
        std*2,
        params
    ) )
